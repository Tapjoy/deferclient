Defer Panic Client Lib.

### Installation
``go get github.com/deferpanic/deferclient``


Get an API KEY:
```
 curl https://api.deferpanic.com/v1/users/create \
        -X POST \
        -d "email=test@test.com" \
        -d "password=password"
```
